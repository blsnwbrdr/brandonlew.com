TIP JAR App

A tipping app with data from countries around the globe. It guides world travelers on the tipping etiquette from different countries and includes currency exchange vs the USD. The web app is built with React, while the iOS app is developed using React Native. The back-end API is built with MongoDB, NodeJS, and Express, hosted on Heroku.

Travel Tracker App

The web app is built with Angular, while the iOS app is developed using React Native. Scroll through 200+ countries/territories and select visited places. Users can view a visual map and list of visited countries/territories. The iOS app includes sharing functionality to allow others to see a user's lists. The back-end API is built with MongoDB, NodeJS, and Express, hosted on Heroku.
