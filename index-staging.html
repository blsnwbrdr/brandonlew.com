TIP JAR App

A tipping app with data from countries around the globe. It guides world travelers on the tipping etiquette from different countries and includes currency exchange vs the USD. The web app is built with React, while the iOS app is developed using React Native. The backend API is built with MongoDB, NodeJS, and Express, hosted on Heroku.

Travel Tracker App

An iOS mobile app written in React Native. Scroll through 200+ countries/territories and toggle visited places. Users can view a visual map and list of visited countries/territories. The app includes sharing functionality to allow others to see a user's list. The back end API is built with MongoDB, NodeJS, and Express, hosted on Heroku.

Etsy App

This app utilizes the Etsy API and pulls data from the Etsy servers to populate the page. The gallery of images, text, and links to Etsy are all pulled from Etsy, making it easy to manage the content and great for businesses that want a customized site while minimizing the amount of work. The app uses Javascript, AJAX, and the Etsy API to create the content.
